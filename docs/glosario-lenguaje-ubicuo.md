# A.1 — Glosario de Lenguaje Ubicuo

| Nombre | Definición | Clasificación |
|------|------|------|
| Solicitud | Representa una petición académica registrada por un usuario que debe ser gestionada dentro del sistema. | Agregado Raíz |
| Usuario | Persona que interactúa con el sistema para registrar o gestionar solicitudes académicas. | Entidad |
| Email | Dirección de correo electrónico asociada a un usuario. Debe tener formato válido. | Value Object |
| TipoUser | Representa el rol de un usuario dentro del sistema. | Value Object |
| TipoSolicitud | Clasificación del tipo de solicitud académica registrada. | Value Object |
| Prioridad | Nivel de urgencia asignado a una solicitud para determinar su orden de atención. | Value Object |
| EstadoSolicitud | Representa el estado actual del ciclo de vida de una solicitud. | Value Object |
| CodigoSolicitud | Identificador único de una solicitud dentro del sistema. | Value Object |
| HistorialSolicitud | Registro auditable de las acciones realizadas sobre una solicitud. | Entidad |
